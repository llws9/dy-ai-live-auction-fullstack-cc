## ⚠️ 已知限制

### 1. 用户认证
**当前状态**: 硬编码用户ID为1
**影响**: 所有出价都以用户ID=1的名义提交
**解决方案**: 需要实现登录/注册功能

### 2. 直播播放器
**当前状态**: 黑色占位符
**影响**: 无法观看真实直播
**解决方案**: 需要集成直播播放器（推荐flv.js）

### 3. 商品图片
**当前状态**: 仅支持URL输入
**影响**: 需要手动提供图片URL
**解决方案**: 需要实现图片上传功能

## 📞 快速测试

### 创建新商品并开始竞拍
```bash
# 1. 创建商品（Admin端）
curl -X POST http://localhost:8081/api/v1/products \
  -H "Content-Type: application/json" \
  -d '{"name": "测试商品", "description": "用于测试"}'

# 2. 配置竞拍规则（会自动创建竞拍）
curl -X POST http://localhost:8081/api/v1/products/6/rules \
  -H "Content-Type: application/json" \
  -d '{"start_price": 0, "increment": 10, "duration": 300}'

# 3. 访问竞拍页面（H5端）
# 打开 http://localhost:5173/auction/6
```
