## 🔧 核心工作流技能

### 1. **base-workflow**
**描述**: 完整研发流程编排
**用途**: 任务创建、IDL变更、编码、部署、测试、收尾全流程
**适用场景**: 从零开始的新需求开发

### 2. **post-coding-verify**
**描述**: 编码后自动验证
**用途**: PPE部署、接口测试、日志验证
**适用场景**: 代码完成后需要快速验证

### 3. **bits-dev-workflow**
**描述**: BITS开发流程
**用途**: 针对BITS平台的开发流程
