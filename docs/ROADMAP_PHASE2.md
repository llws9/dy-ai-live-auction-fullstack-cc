## 二、核心功能增强

### 2.1 多语言国际化支持 🌍

**优先级**: P0
**预估工期**: 3周

#### 功能描述
为系统提供完整的多语言支持,支持中英文切换,为后续拓展其他语言预留接口。

#### 技术方案

**前端**:
- 使用 `react-i18next` 作为国际化框架
- 语言包管理:
  ```
  locales/
  ├── zh-CN/
  │   ├── common.json      # 通用文本
  │   ├── auction.json     # 竞拍相关
  │   ├── product.json     # 商品相关
  │   └── notification.json # 通知相关
  └── en-US/
      ├── common.json
      ├── auction.json
      ├── product.json
      └── notification.json
  ```

**后端**:
- 使用 `go-i18n` 库
- 根据请求头 `Accept-Language` 返回对应语言
- 数据库多语言字段设计:
  ```sql
  -- 商品表多语言字段
  ALTER TABLE products ADD COLUMN name_i18n JSON;
  ALTER TABLE products ADD COLUMN description_i18n JSON;

  -- 示例数据
  {
    "zh-CN": "商品名称",
    "en-US": "Product Name"
  }
  ```

**API接口预留**:
```go
// 获取多语言内容
type I18nContent struct {
    Lang    string `json:"lang"`
    Content string `json:"content"`
}

// 商品多语言字段
type ProductI18n struct {
    Name        []I18nContent `json:"name"`
    Description []I18nContent `json:"description"`
}
```

**语言切换流程**:
1. 前端检测浏览器语言或用户设置
2. 通过 `Accept-Language` 请求头传递语言偏好
3. 后端返回对应语言的文本内容
4. 数据库中的多语言字段按语言返回

**实施步骤**:
1. Week 1: 后端i18n框架集成、数据库字段扩展
2. Week 2: 前端i18n框架集成、语言包编写
3. Week 3: 全站文本国际化、测试验证

**验收标准**:
- ✅ 支持中英文切换
- ✅ 所有用户可见文本国际化
- ✅ API支持语言参数
- ✅ 语言包可扩展
