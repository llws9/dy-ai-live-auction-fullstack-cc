## 🎮 完整演示流程

### 第一步：管理后台配置

1. **访问管理后台**: http://localhost:5175
2. **登录**（测试账号）:
   - 用户名: `admin`
   - 密码: `admin123`

3. **创建商品**:
   - 进入"商品管理"
   - 点击"创建商品"
   - 填写商品信息（名称、描述、图片）
   - 保存

4. **配置竞拍规则**:
   - 在商品列表中点击"配置规则"
   - 设置竞拍参数:
     - 起拍价: 0元
     - 加价幅度: 10元
     - 竞拍时长: 300秒（5分钟）
     - 封顶价: 1000元（可选）
   - 保存后自动创建竞拍场次

5. **查看数据**:
   - 数据大屏: 实时监控
   - 统计报表: 详细数据

---

### 第二步：用户参与竞拍

1. **访问用户端**: http://localhost:5173
2. **注册/登录**（可注册新用户或使用测试账号）:
   - 用户名: `user1` / `user2` / `user3`
   - 密码: `123456`

3. **浏览竞拍**:
   - 查看首页推荐
   - 进入竞拍列表
   - 点击查看详情

4. **参与出价**:
   - 点击"出价"按钮
   - 输入金额（需符合加价规则）
   - 观察实时排名更新

5. **竞拍结束**:
   - 查看中标通知
   - 在用户中心查看订单

## 📞 如遇问题

- 检查端口是否被占用: `lsof -i :5173` / `lsof -i :5175`
- 查看日志: `/tmp/h5.log` / `/tmp/admin.log`
- 重启服务: 停止进程后重新运行 `npm run dev`
